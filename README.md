# 📚 Library Management System (C++)

A console-based Library Management System built using C++ that demonstrates Object-Oriented Programming (OOP), CRUD operations, searching, updating, and a colorful console interface.

---

## ✨ Features

- ➕ Add New Book
- 📖 Display Books
- 🔍 Search Book by ID
- ✏️ Update Book Details
- ❌ Delete Book
- 📊 Count Total Books
- 🎨 Colorful Console Interface

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming
- Arrays
- Windows Console API
- MinGW GCC

---

## 🚀 How to Run

```bash
g++ LibraryManagement.cpp -o LibraryManagement
LibraryManagement.exe
```
